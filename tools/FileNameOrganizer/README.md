# FileNameOrganizer

This folder contains the macOS self-contained single-file executable for the "FileNameOrganizer" project.

Built by GitHub Actions on push. To run locally on macOS:

1. Download the executable from this repository.
2. Make it executable: `chmod +x FileNameOrganizer`
3. Run: `./FileNameOrganizer`

If you need a different runtime (arm64) or a non-single-file publish, adjust the workflow.
